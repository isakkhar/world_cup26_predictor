# 🏆 2026 World Cup Bracket Predictor

A high-end, interactive web application designed to simulate and predict the results of the 2026 FIFA World Cup. Featuring the new 48-team format, this predictor handles complex group-stage mathematics and knockout bracket generation with a premium "Sports Dashboard" aesthetic.

![Landing Page Preview](https://placehold.co/1200x600/020617/FFFFFF/png?text=2026+World+Cup+Predictor+by+Sakkhar+Saha)

## ✨ Key Features

- **New 48-Team Format**: Fully implements the 12-group stage structure for the 2026 World Cup.
- **Dynamic 3rd Place Logic**: Automatically calculates the 8 best 3rd-place teams to complete the Round of 32, handling all 495 possible combinations.
- **Interactive Knockout Bracket**: Visualize the entire journey from the Round of 32 to the Final at MetLife Stadium.
- **Premium UI/UX**: Cinematic "Glassmorphic" design with neon accents, smooth animations, and high-contrast accessibility.
- **Golden Trophy Podium**: A regal celebration screen for the predicted champion with silver and bronze medalists.
- **Smart Progress Tracking**: Real-time progress bars and match counters (32 matches total in the knockout phase).
- **Social Sharing**: Share your ultimate tournament prediction with friends via a customized link or message.

## 🛠️ Tech Stack

- **Framework**: React 18+
- **Build Tool**: Vite
- **Language**: TypeScript (Type-safe bracket logic)
- **Styling**: Vanilla CSS (Custom Glassmorphism & Modern Utility Classes)
- **Icons**: Lucide React
- **Deployment**: Optimized for Vercel / Netlify

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/wc26-predictor.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```

## 📂 Project Structure

- `/src/data`: Contains tournament data and bracket structure logic.
- `/src/pages`: Main application components (Landing Page, Predictor).
- `/src/components`: Reusable UI components like Modals and Cards.
- `/src/assets`: Images and global styles.
